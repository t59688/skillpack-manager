Command Reference

Full command set for the complete skill pack lifecycle.

skillpack scan [path] P0

Scan existing skill directories in common agent paths. Shows present, missing, and modified skills.

skillpack scan ~/.claude/skills
skillpack create <name> P0

Create a new skill pack with manifest, owner namespace, and visibility settings.

skillpack create sales-pack --owner tiechui
skillpack add <pack> <skill> P0

Add skills from agent directories to your pack. Interactive picker for multi-select.

skillpack add ./sales-pack ~/.claude/skills/customer-summary
skillpack pack <dir> P0

Create a distributable .skillpack archive with manifest, skills, and shared resources.

skillpack pack ./sales-pack --out dist
skillpack install <source> P0

Install a pack into one or more agent targets. Records installation metadata for tracking.

skillpack install ./dist/sales-pack.skillpack --target claude
skillpack publish <dir> P1

Publish to GitHub Releases or local directory. Supports version bumping and overwrite control.

skillpack publish ./sales-pack --to github --repo owner/repo
skillpack pull <source> P1

Pull a published pack from GitHub or remote registry into a local workspace for editing.

skillpack pull github:owner/repo --out ./sales-pack
