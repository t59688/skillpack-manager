Everything you need

From scanning skills to publishing them — a complete lifecycle for your AI agent configurations.

Scan

Discover skills in common agent directories. Interactive checkbox picker for multi-select scanning.

skillpack scan ~/.claude/skills

Create

Package multiple skills into a reproducible stack with a YAML manifest and version control.

skillpack create sales-pack

Pack

Bundle your pack into a portable .skillpack artifact with checksums for integrity verification.

skillpack pack ./sales-pack --out dist

Install

Install packs into any supported agent with automatic directory mapping and safe uninstallation.

skillpack install ./pack.skillpack --target claude

Audit

Static analysis for security — detects suspicious scripts, embedded secrets, and destructive commands.

skillpack audit ./sales-pack

Diff & Sync

Detect drift between pack and installed skills. Sync changes across machines and team members.

skillpack diff ./sales-pack --target claude
